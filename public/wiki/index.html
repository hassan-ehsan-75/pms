API tokens

An API token provides an easy way to authnticate to the the nasable inside system for fetching public data. To generate the API token you need to provide a secret key/password and the framework name in which the API will be used by.

News API

The News API allows to get all the latest "PUBLISHED" aricles in the nasable inside system. To fetch the latest news you need first to generate a token to be able to call the api from a specific framework. To start you need to call the link https://website_url.com/api/getNews and send two parameters token and count.

Call paramters
ParameterDiscription
tokenThe generated api token for your framework
countA limit on the articles to be returned
Return paramters
ParameterDiscription
titleArticle's title
contentArticle's content including HTML tags
summaryArticle's summary (the first 200 chracters of the contetnt) with HTML tags striped
created_atThe timestamp when the aricle was first created
updated_atThe last time the article was updated
attachmentimage name and extention that resides in the cdn directory

Users API

The Users API allows to get all users in the nasable inside system. To start you need to call the link https://website_url.com/api/getUsers and send one parameter token.

Call paramters
ParameterDiscription
tokenThe generated api token for your framework
Return paramters
ParameterDiscription
full_nameUser's full name
titleUser's title in the company
emailUser's email address
twitterTwitter id
linkedinLinkedIn id
aboutLong text about the user
attachmentUser's photo: name and extention that resides in the cdn directory
